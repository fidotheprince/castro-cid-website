Alvaro's Porfolio
The goal when designing this portfolio can be broken down in two words, display case. Accordingly the layout, designed in Sketch, is meant to visually unobstruct the user from directing their eyes to the projects immediately. Working hand-in-hand with said design, the user navigates the website with a minimal number of clicks. This saves the user time, and provides a quick/intuitive understanding of the site.

HTML5 is used to present content. A specific CSS3 layout model named Flexbox was used to position all elements of the site. This same model helps with the site's viewport responsiveness. For the dynamic elements, there is a sticky navigation bar integrated into the site through plain JavaScript.

Scalability and simplicity were key drivers when assessing the technology stack for the portfolio. The goal is to avoid over engineering the basic purpose of the site. That being said, the site has room for improvement. In particular, adding a handful of dynamic elements, like a hamburger menu for mobile and perhaps a photo gallery for each project.
